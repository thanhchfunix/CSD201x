JDK version: 13 (13.0.1)
IDE: eclipse(ver 4.14.0)

1. Cài đặt chương trình:
	- Chạy công cụ eclipse.
	- Import file project (File -> Open Projects from file System)
2. Chạy chương trình:
	- Tab Project Explorer -> file project CSD201x-A2 
	-> file src ->default pagkage, run file Main.java
	- Menu:
		1. Input book and add to the end list
		2. Display all book of list
		3. Search book by code
		4. Add book after position k
		5. Delete book at position k
		6. Sort book by code (ascending)
		7. Sort book by price (ascending)
		8. Exit
