Project 1: Hệ thống quản lý tập tin
JDK version: 13 (13.0.1)
IDE: eclipse(ver 4.14.0)

1. Cài đặt chương trình:
	- Chạy công cụ eclipse.
	- Import file project (File -> Open Projects from file System)
2. Chạy chương trình:
	- Tab Project Explorer -> file project asignment2_PRO192x_cathuythanh -> file src -> pagkage ui, run file PRO192xA2Solution.java
	- Menu: 1: Tạo list random 10 con ong trong đó có gồm 3 loại: QueenBee, WorkerBee, Drone.
						list có duy nhất 1 QueenBee.
					2: Tấn công những con ong còn sống (làm giảm health sau mỗi lần tấn công) cho đến khi chúng chết (hiển thị "dead"), khi đã chết nếu tiếp tục nhập 2, thuộc tính health sẽ không tiếp tục bị giảm.
					Có thể tấn công đến khi tất cả 10 con ong cùng chết (hiển thị "All bees are dead!")
					Có thể nhập lại 1 hoặc 2 nhiều lần
					3: Exit.
